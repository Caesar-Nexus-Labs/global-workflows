# Premium UI/UX Design (/forge-design)

## Caesar Nexus Execution Logic

### Phase 1: Visual Research
- **Call**: `/forge-brainstorm`
- **Action**: Gather inspiration and define color palettes (Glassmorphism, Dark Mode).

### Phase 2: Blueprinting
- **Call**: `/forge-plan`
- **Action**: Orchestrate component breakdown and responsive layouts.

### Phase 3: Execution (Asset Generation)
- **Call**: `generate_image`
- **Action**: Create icons, mockups, and illustrative assets.

### Phase 4: Prototype & Review
- **Call**: `/forge-cook`
- **Action**: Build interactive components via CSS/Vite/NextJS.

### Phase 5: Finalization
- **Verify**: Cross-browser compatibility and accessibility (A11y) check.

## Iron Laws
- **Zero Placeholder**: No generic images; all assets must be generated or provided.
- **Premium Aesthetics**: Mandatory use of rich gradients and micro-animations.
- **Performance**: UI must achieve Lighthouse score ≥ 90.
