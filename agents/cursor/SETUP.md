# Setup — Cursor

## Installation

1. Clone this repo:
```bash
git clone https://github.com/Caesar-Nexus-Labs/caesar-builder.git
```

2. Add to your `.cursorrules` or `AGENTS.md`:
```markdown
# Caesar Nexus Workflows
Slash commands available in: /path/to/caesar-builder/commands/
Use /forge, /forge-plan, /forge-debug, /ops-deploy, etc. as described in each .md file.
Triggers (quality gates): /path/to/caesar-builder/triggers/
```

3. Reference specific commands in Cursor chat by pasting the command content or pointing to the file path.

## Commands available
See [commands/](../../commands/) — 105 slash commands across 6 kits.
