# Marketing Dashboard Workflow (/market-dashboard)

## Caesar Nexus Execution Logic

### Phase 1: Metric Selection
- **Action**: Identify critical KPIs and set target goals.
- **Goal**: Define calculations and data sources.

### Phase 2: Integration & Setup
- **Action**: Connect Google Analytics, Ad platforms, and CRM via `/market-analytics`.
- **Verify**: Data freshness and API connection stability.

### Phase 3: Design & Layout
- **Action**: Orchestrate the dashboard UI following the Caesar Nexus premium design.
- **Action**: Implement drill-down capabilities for deep-dive analysis.

### Phase 4: Sharing & Access
- **Action**: Configure user permissions and scheduled reports.
- **Verify**: Stakeholder access and automated delivery.

### Phase 5: Maintenance & Optimization
- **Action**: Weekly accuracy verification and bug fixing.
- **Action**: Monthly refinement based on business needs.

## Iron Laws
- **Real-Time Accuracy**: Data must be refreshed at least daily.
- **Visual Impact**: Use clean, modern visualizations (No "Chart Junk").
- **Privacy Gated**: Restrict sensitive financial data to authorized personnel only.
