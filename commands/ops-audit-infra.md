# Infra Audit & Hardening Workflow (/ops-audit-infra)

## 1. What This Workflow Does
Security and compliance sweep for the infrastructure layer.

## 2. Actions
- **Port Scan**: Ensures no unauthorized ports are open.
- **IAM Audit**: "Least Privilege" verification for service accounts.
- **Vulnerability Scan**: Checks OS and container images for CVEs.
