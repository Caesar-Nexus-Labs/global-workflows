# Problem Solving Workflow (/forge-problem-solving)

## 1. What This Workflow Does
A generalized framework for tackling non-code problems (e.g., architectural bottlenecks, logic paradoxes). 

## 2. Methodology
1. **Define**: Isolate the core contradiction.
2. **Deconstruct**: Break the problem into atomic variables.
3. **Simulate**: Mentally (AI-side) run 3-5 scenarios to find the path of least resistance.
4. **Solve**: Implement the solution and verify zero side-effects.
