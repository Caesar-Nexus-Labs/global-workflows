# /market-write

> **High-Conversion Copywriting** — Strategic messaging and persuasive content for all channels.

## Metadata

- **Slug**: /market-write
- **Type**: Content Creation
- **Agents**: Copywriter, Messaging Strategist

## Description

Creates high-converting copy for:
- Sales pages & landing pages
- Email sequences
- Ad copy (Google, Facebook, LinkedIn)
- Social media captions
- Product descriptions
- CTAs & messaging hierarchy

Incorporates psychological principles, A/B testing frameworks, and data-driven insights.

## Parameters

| Flag | Type | Description |
|------|------|-------------|
| --content-type | string | sales-page, email, ad-copy, landing-page |
| --target-audience | string | Audience segment & persona |
| --primary-benefit | string | Core value proposition |
| --cta-goal | string | Newsletter signup, free trial, purchase, etc |
