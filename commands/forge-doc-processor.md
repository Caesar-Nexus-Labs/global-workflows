# Documentation Processor (/forge-doc-processor)

## Caesar Nexus Execution Logic

### Phase 1: Content Ingestion
- **Action**: Batch capture raw files or crawl web documentation (No browser tool).

### Phase 2: Structured Analysis
- **Action**: Identify key logic, iron laws, and step-by-step processes.

### Phase 3: Synthesis & Whitewash
- **Action**: Transform raw data into Caesar Nexus standardized Markdown.

### Phase 4: Verification
- **Action**: Check against original source for logic "tightness" and accuracy.

### Phase 5: Distribution
- **Update**: Commit refined docs to `skills_repo/` or project `docs/`.

## Iron Laws
- **High Fidelity**: Never summarize critical logic; maintain absolute fidelity to the source "ruột".
- **Brand Purity**: Always remove external branding and links.
- **Traceability**: Maintain internal references to the original ingestion timestamp.
